# 로또 번호 추천 사이트

## 개요

이 프로젝트는 사용자에게 로또 번호를 추천해주는 웹 애플리케이션입니다. 사용자는 버튼을 클릭하여 6개의 랜덤 번호를 생성하고 화면에서 확인할 수 있습니다.

## 스타일 및 디자인

- **색상 팔레트:**
  - 배경: `oklch(0.98 0.02 240)`
  - 기본 텍스트: `oklch(0.2 0.02 240)`
  - 강조색: `oklch(0.6 0.15 260)`
- **타이포그래피:**
  - 제목: 'Orbitron', sans-serif (Google Fonts)
  - 본문: 'Noto Sans KR', sans-serif (Google Fonts)
- **레이아웃:**
  - 중앙 정렬 컨테이너
  - 부드러운 그림자 효과와 둥근 모서리를 가진 카드 UI
- **상호작용:**
  - 버튼 클릭 시 부드러운 애니메이션 효과

## 기능

- **로또 번호 생성:**
  - 1부터 45까지의 숫자 중에서 6개의 고유한 숫자를 랜덤으로 생성합니다.
- **결과 표시:**
  - 생성된 번호를 시각적으로 매력적인 방식으로 화면에 표시합니다.
  - Web Components를 사용하여 번호 표시 부분을 캡슐화합니다.
